如何使用hexo-theme-cola主题

hexo-theme-cola是基于Hexo的博客主题，其风格类型主要是：简洁。

没有花哨的界面，没有复杂的功能。主要就是为了使用者能更聚集的编写博客。

开始

首先通过Hexo初始化项目：

hexo init blog

克隆完成后，进入项目，并且克隆主题到themes下：

git clone git@github.com:Aizener/hexo-theme-cola.git

完成以上步骤后，就可以开始配置根目录下的_config.yml文件，一共需要配置两个地方：

  • 关闭highlight配置项，打开prismjs配置：

    highlight:
  enable: false    # true改为false
  line_number: true
  auto_detect: false
  tab_replace: ''
  wrap: true
  hljs: false
prismjs:
  enable: true    # false改为true
  preprocess: true
  line_number: true
  tab_replace: ''

  • 把theme配置改为：hexo-theme-cola主题：

    theme: hexo-theme-cola

完成以上步骤后，我们再继续安装所需要的包依赖。

安装完成后，执行以下命令重新生成主题，并开启服务。

hexo clean & hexo g & hexo s

新建页面

安装好主题后，还需要新建几个页面，分别执行以下命令：

  1. 新建日志页面：hexo new page log

  2. 新建友链页面：hexo new page link

  3. 新建个人页面：hexo new page about

  4. 新建分类页面：hexo new page categories

  5. 新建标签页面：hexo new page tags

  6. 新建工具页面：hexo new page tools

新建完成页面之后，就可以到主题的_config.yml配置文件配置对应的信息了。（页面根据是否需要来新建）

日志页面需要通过命令新建一个文件：

hexo new page --path log/logName "logName"

执行该命令后，在log目录下会新增一个title为logName的文件（改成自己想要的标题即可），然后像写文章一样即可编写日志。

像上面这样的格式编写即可，日志页面就会以从上到下的顺序显示了。

杂项修改

可以看到，网站上还有些地方的配置没有介绍，配置这些也非常简单。进入主题目录找到_config.yml文件，里面有已经配置好的项，照着替换即可。

title: 可乐的小站
# 左侧栏名字
author_name: Cola.

# 左侧栏标签，只能两个
tag1: 中二
tag2: 宅

# 左侧栏Github连接
github_url: https://github.com/Aizener

# 左侧栏座右铭，分两句
motto_prev: “花有重开日，人无再少年”
motto_last: “一个简单普通的男孩”

# 小站部署日期（运营时间）
public_date: 2022-02-16

# 左侧栏菜单
menu:
  - [小站首页, '/', 'icon-shouye']
  - [个人日志, '/log', 'icon-rizhi']
  - [友情链接, '/link', 'icon-youqinglianjie']
  - ['关于自己', '/about', 'icon-guanyuwomen']

# 友情连接：头像、名字、链接
links:
